# Architecture Diagrams

## Shape Cheatsheet

  | Shape | Name | Means |
  |-------|------|-------|
  | ☁️ Cloud | Cloud/Blob | External service or remote API (outside your system) |
  | 🗄️ Cylinder | Database | Persistent storage / database table |
  | ⬜ Rectangle | Process/Service | Internal service, worker, or process |
  | ➡️ Chevron/Arrow | Output/Sink | Data being written or pushed to storage |
  | ◇ Diamond | Decision | Branch / conditional logic |
  | ⬭ Oval/Ellipse | Terminal | Start or end of a flow |
